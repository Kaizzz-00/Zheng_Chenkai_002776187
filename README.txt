The project is a Hospital Information System with multiple users.

User are divided into 3 groups:Patients,Doctors,Admins, and Admins contains SysAdmin,HosAdmin,ComAdmin

20 classes of Model were created in this file, including classes and their Directories.
18 Frames and Panels were created for different functions of different users.

The data storage of the project is based on ArrayList and passing classes between frames and panels.
The workFlow is described in the sequence diagram and the class components are displayed in Class diagram
